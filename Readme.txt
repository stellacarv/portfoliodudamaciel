# Portfolio | Duda Maciel 📸💻

Repositório do portfólio profissional de Duda Maciel, desenvolvido com foco em performance, design minimalista e experiência do usuário (UX/UI). O projeto apresenta uma interface dual, destacando a atuação como modelo internacional e as competências técnicas em desenvolvimento front-end.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5 & CSS3:** Estruturação semântica e estilização customizada (Mobile First).
* **JavaScript:** Implementação de interatividade e manipulação de DOM.
* **Figma:** Prototipação de alta fidelidade e definição do Guide de Estilo.
* **Git & GitHub:** Controle de versão e deploy.

## 🎨 Design & UX

O design foi concebido para transmitir sofisticação e clareza técnica:
* **Color Palette:** Uso da regra 60-30-10 para equilíbrio visual.
* **Dark Mode:** Interface focada em destacar o conteúdo visual (fotografias).
* **Responsividade:** Adaptado para múltiplos breakpoints (Desktop, Tablet e Mobile).

## 📁 Estrutura da Seção 'Informações Pessoais'

Um dos diferenciais deste portfólio é a seção técnica de casting, onde os dados físicos são apresentados de forma direta. O texto de apoio foi otimizado para uma abordagem B2B:

> "Dados técnicos e parâmetros físicos para análise de casting. Estrutura versátil para diferentes demandas criativas, focada em otimizar o fluxo de trabalho entre modelo e marca através de profissionalismo e prontidão técnica."
